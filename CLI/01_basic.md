# Basic CLI

## 폴더(directory) 관련

폴더 상징 특수기호

- `~`
  - Home 폴더를 의미
  - 위치는 일반적으로 /C/Users/developer_minki
- `/`
  - Root 폴더를 의미
  - 최상단을 의미
- `.`
- `..`



## 명령어

- `cd`

- `ls`

- `rm`

- `mkdir`

- `touch`

  ```bash
  mkdir TIL
  cd TIL
  mkdir CLI
  cd CLI
  touch 01_basic.md
  ```
